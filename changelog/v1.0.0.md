# RxFHIR v1.0.0

## Added

- Initial Electron desktop application for creating and querying FHIR R4 electronic prescription bundles.
- Creator workflow for Organization, Patient, Practitioner, Encounter, Condition, Observation, Coverage, Medication, MedicationRequest, Extension, and Composition.
- Consumer workflow with basic, date, and complex search modes, structured detail view, and raw JSON viewer.
- Settings page with configurable FHIR server presets, health check, theme toggle, and bilingual interface.
- GitHub Actions workflow for automated macOS and Windows builds.
- Windows release artifacts now include both an installer and a portable executable.

## Changed

- Release notes are now version-scoped and sourced from `changelog/vX.Y.Z.md`.
