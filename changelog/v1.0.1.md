# RxFHIR v1.0.1

## Changed

- Updated the GitHub Actions workflow to use currently supported runner labels for macOS and Windows.
- Upgraded core GitHub Actions steps to Node 24 compatible versions.
- Made macOS and Windows release builds use explicit targets and x64 architecture for more predictable packaging.
- Added `author` metadata to `package.json` to reduce electron-builder warnings during CI builds.

## Fixed

- Fixed the unsupported macOS runner configuration in the release workflow.
- Reduced the risk of Windows release build failures caused by runner image drift and implicit architecture selection.
