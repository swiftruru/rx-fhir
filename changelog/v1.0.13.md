# RxFHIR v1.0.13

## Added

- Added a full keyboard shortcut system with a centralized help dialog and a Settings page for shortcut customization, conflict detection, and restoring defaults.

## Changed

- Added common outpatient prescription templates through a compact toolbar entry instead of a large inline panel.
- Refined the Feature Showcase experience with stronger spotlight focus, darker background dimming, nearby coaching panels, and more polished navigation controls.

## Fixed

- Fixed the shortcut help dialog layout so it stays within the viewport, supports internal scrolling, and avoids large empty sections.
- Fixed Creator stepper scrolling so later steps such as `Document Bundle` remain reachable.
- Fixed Consumer query-step explanations so they follow the active UI language instead of leaving Chinese text in English mode.
- Fixed Settings connection testing so a successful check immediately updates the current connection status shown below and in the status bar.
