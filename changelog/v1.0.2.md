# RxFHIR v1.0.2

## Added

- Embedded `Noto Sans TC` into the renderer build to improve offline typography consistency, especially on Windows.
- Added active-tab aware recent-record prefills for Consumer basic, date, and complex search flows.
- Added form backfill for completed Creator steps so revisiting a resource restores its current values.

## Changed

- Updated Consumer complex search to use HAPI-compatible client-side filtering for physician author queries.
- Updated the About page to list advisors in alphabetical order by English name and explicitly note that no ranking is implied.
- Refined README documentation to match the current Consumer search behavior, embedded font setup, and Creator editing workflow.

## Fixed

- Fixed repeated Creator submissions creating duplicate resources after revisiting a completed step.
- Fixed Coverage updates failing with duplicate `POST` errors when the step had already been created once in the same workflow.
- Fixed Consumer author search failing to match existing practitioner names because of unresolved Bundle author references.
