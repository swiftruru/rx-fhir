# RxFHIR v1.0.7

## Added

- Added step-level completion summaries in the Creator sidebar and current-step header so completed content is easier to review without reopening each form.
- Added richer inline guidance and examples for higher-friction fields such as ICD-10 codes, LOINC codes, medication frequency and route, and supplemental extension inputs.
- Added persistent Consumer search history with separate recent-search and favorite-search sections for faster reruns.

## Changed

- Updated the Consumer empty-result experience to explain likely causes and suggest next actions based on the actual search mode and query trace.
- Updated the Creator JSON preview with font-size controls, collapse-all support, and a latest-resource-only filter to improve demos and walkthroughs.
