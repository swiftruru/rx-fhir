# RxFHIR v1.0.8

## Added

- Added Linux release artifacts to GitHub Actions, including `AppImage` and `deb` packages.
- Added a dedicated Feature Showcase demo mode with spotlight highlighting, guided coach content, and presentation-oriented transitions.
- Added a production-style Live Demo flow with manual-first pacing, optional autoplay, simulated typing, and collapsible demo controls.

## Changed

- Expanded the locale-aware Mock Data System so the same demo scenarios can resolve to Chinese or English content while keeping shared identifiers, codes, dates, and other base fields stable.
- Improved the FHIR request inspector with clearer request-flow explanations, clickable URLs, and a compact `GET / POST / PUT` quick guide.
- Reworked the Consumer query workspace so import and query examples now live under `Query Helpers`, keeping the main search form focused on real search tasks.
- Updated README with the latest feature set, screenshots, demo capabilities, and cross-platform release information.

## Fixed

- Fixed multiple public HAPI duplicate-resource edge cases by improving fallback reuse behavior after duplicate `POST` failures.
- Fixed recent-submission prefills and complex-query helper behavior so full Bundle records can backfill search context more reliably.
- Fixed several Feature Showcase overlay and coach positioning issues, including sidebar sizing, spotlight transitions, and background masking consistency.
